Book End Privacy Policy

This document outlines how the application Book End makes use of your data. It also explains how a user can delete / revoke any data.

On device data

When using the application, information about books and reading sessions are stored on your device locally. These remain private (on device) and are never transfered to any external servers.

The following information is stored on your device:

Books

  • Author
  • Book Description
  • Genre
  • Image Cover
  • Pages Current
  • Pages Total
  • Title

Reading Sessions

  • Start Time / Date
  • End Time / Date
  • Pages read during session
  • Reading speed (pages per hour)
  • Duration of reading session

Scanning Books

When scanning a book's ISBN / Barcode using the "Scan Book" feature, a text copy of the ISBN is sent as a request to Google Books. This is used to return a books data. This data is sent privately, and at no time is the developer able to view or access any data sent, or view any results from the camera. No data of this request is stored.

Photo Access

Photo Library access is required in order to import images of books into the app. No image data is ever transfered off your device.

To revoke this, please do so in the iOS Privacy Settings.

Removing Data

To remove indivual data, please delete the associated books / reading sessions from your device

To delete ALL DATA please delete the entire app.
